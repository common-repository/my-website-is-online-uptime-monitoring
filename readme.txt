= My Website is Online features =

Receive a notification as soon as a problem is detected on your website.

* 1-minute checks
* Unlimited monitors
* SMS alerts
* Unlimited email alerts
* HTTP headers and status
* Response time monitoring (TTFB)
* SSL certificate checks
* Multi-factor authentication
* 12 months log retention
* Multiple email recipients
* Domain expiration monitoring
* Google Web Risk
* DNS records monitoring
* Slack integration

To learn more about My Website is Online visit: [https://mywebsiteisonline.com/](https://mywebsiteisonline.com/)

== Installation ==

1. Extract `mwio-uptime.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Insert your website code into the input field on the configuration page
1. Validate your code and start monitoring your website!
